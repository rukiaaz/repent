# Privacy Policy

**Last Updated:** June 11, 2026

## 1. Introduction

Welcome to Repent ("the Bot," "we," "us," or "our"). This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our Discord bot. Please read this privacy policy carefully. If you do not agree with the terms of this privacy policy, please do not use the Bot.

## 2. Information We Collect

### 2.1 Discord-Provided Information
When you add Repent to your Discord server, we collect the following information provided by Discord:
- Server ID and name
- User IDs and usernames of server members
- Channel IDs and names
- Role IDs and names
- Message content (for automod and security features)
- Audit log entries (for antinuke protection)
- Voice activity data (when relevant to security monitoring)

### 2.2 Security-Related Data
For the purpose of providing security and moderation features, we collect:
- User punishment records (bans, kicks, timeouts)
- Whitelist entries (user, bot, and role whitelists)
- Antinuke violation logs
- Raid detection data
- Verification status and timestamps
- Backup snapshots of server configuration

### 2.3 Optional User-Provided Information
- Custom verification messages and embeds
- Welcome and farewell message templates
- Configuration preferences
- Webhook URLs for notifications (if provided)

## 3. How We Use Your Information

We use the collected information for the following purposes:

### 3.1 Core Functionality
- **Security Protection:** Detect and prevent unauthorized actions, nukes, raids, and security threats
- **Moderation:** Enforce server rules through automated moderation and punishment systems
- **Verification:** Manage user verification processes and access control
- **Backup:** Create and restore server configuration backups
- **Logging:** Maintain security and moderation logs for server administrators

### 3.2 Service Improvement
- Analyze usage patterns to improve bot performance and security effectiveness
- Debug technical issues and improve system reliability
- Develop new security features and moderation tools

### 3.3 Communications
- Send security alerts and notifications to server administrators
- Provide essential service updates and maintenance notices
- Respond to support requests and inquiries

## 4. Data Storage and Security

### 4.1 Storage Duration
- **Active Server Data:** Retained while the bot is active in your server
- **Security Logs:** Retained for 30 days, unless required for security investigations
- **Backup Snapshots:** Retained according to user-configured retention periods
- **Punishment Records:** Retained indefinitely unless manually removed
- **Audit Data:** Automatically purged after 90 days

### 4.2 Data Security
We implement appropriate technical and organizational measures to protect your information:
- SQLite database with encryption for sensitive data
- Access controls and authentication for database operations
- Regular security updates and vulnerability assessments
- Secure data transmission using HTTPS/TLS

## 5. Data Sharing and Disclosure

We do not sell, rent, or trade your personal information. We may share information only in the following circumstances:

### 5.1 With Server Administrators
- Security logs and violation reports relevant to their server
- Configuration and settings data for server management
- Backup data for server restoration purposes

### 5.2 With Discord
- When required by Discord's Terms of Service or for bot verification
- To comply with legal obligations or protect Discord's services

### 5.3 Legal Requirements
- When required by law, court order, or government regulation
- To protect our rights, property, or safety
- To prevent fraud or security threats

### 5.4 Business Transfers
- In the event of a merger, acquisition, or sale of assets

## 6. Your Rights and Choices

You have the following rights regarding your information:

### 6.1 Access and Review
- Request access to data stored about you or your server
- Review security logs and moderation records

### 6.2 Data Deletion
- Request deletion of server data by removing the bot from your server
- Request deletion of specific data through support channels
- Data is automatically deleted when the bot leaves a server

### 6.3 Data Correction
- Request correction of inaccurate information
- Update configuration and preferences through bot commands

### 6.4 Opt-Out
- Disable specific features through bot configuration
- Adjust data retention settings where available
- Remove the bot from your server to stop all data collection
